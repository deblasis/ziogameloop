# ziogameloop

> Fixed-timestep game loop with variable rendering. Accumulator-based update, frame rate control.

Part of the [zio-zig](https://github.com/deblasis/zio-zig) ecosystem — a collection of Zig libraries for building games and CLI tools.

## Quick start

```zig
const ziogameloop = @import("ziogameloop");
```

```bash
zig build test      # Run tests
zig build run-example  # Run example
```
